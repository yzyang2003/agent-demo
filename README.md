# AI Agent Demo - 企业级 AI 助手作品集

> 🎯 针对 **湖南向上国际旅行社** 和 **健萌 WEGYMER** 两家企业的求职作品集，展示 RAG、Tool Calling、记忆机制、Agent 推理等核心 AI 能力

## 项目简介

这是一个面向真实企业场景的 AI 助手应用。项目包含两个定制化业务场景：

- **旅游助手**：为湖南向上国际旅行社定制，覆盖长沙旅游的景点、美食、签证、行程规划
- **健身顾问**：为健萌 WEGYMER 定制，覆盖 BMI 评估、卡路里计算、训练计划、营养建议

两个场景共享同一套 Agent 架构，通过 RAG 检索、7 个工具调用、多轮记忆、多步推理，展示从 Prompt 工程到 Agent 系统设计的完整能力链。

## 功能特性

### 🏖️ 旅游助手（向上国际旅行社定制）

- 长沙景点、美食、交通等信息 RAG 检索
- 天气查询、智能行程规划、签证信息查询
- 多轮对话中记住用户偏好（出发地、出行人数等）

### 💪 健身顾问（健萌 WEGYMER 定制）

- 增肌、减脂、营养等知识问答
- BMI 计算、卡路里估算、训练动作推荐、营养素查询
- 根据用户身体数据给出个性化建议

### 🔧 7 个工具

| 工具 | 文件 | 功能 |
|------|------|------|
| 天气查询 | `weather.py` | 查询指定城市天气 |
| BMI 计算 | `bmi.py` | 根据身高体重计算 BMI |
| 行程规划 | `itinerary.py` | 生成多日行程方案 |
| 签证查询 | `visa.py` | 查询目的地签证政策 |
| 卡路里估算 | `calorie.py` | 估算食物或运动热量 |
| 训练推荐 | `workout.py` | 推荐训练动作和计划 |
| 营养素查询 | `nutrition.py` | 查询食物营养成分 |

### 🧠 记忆机制

`MemoryManager` 管理多轮对话的上下文记忆：

- **短期记忆**：当前会话的对话历史，保留最近 N 轮
- **实体记忆**：从对话中提取关键信息（用户偏好、身体数据等），跨轮次持久化
- **记忆注入**：推理时将记忆摘要注入 Prompt，让模型"记住"用户

### 🤖 Agent 推理

`AgentEngine` 实现多步推理循环：

1. 接收用户输入 + 记忆上下文
2. 模型判断是否需要调用工具
3. 执行工具，将结果反馈给模型
4. 模型整合信息，生成最终回答
5. 全程展示推理步骤（思维链 + 工具调用过程）

## 本地运行

### 1. 克隆项目

```bash
git clone https://github.com/yz/agent-demo.git
cd agent-demo
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key
```

### 4. 运行应用

```bash
streamlit run app.py
```

访问 http://localhost:8501 即可使用。

## 部署到 Streamlit Cloud

1. Fork 本仓库到你的 GitHub 账号
2. 访问 [Streamlit Cloud](https://share.streamlit.io)
3. 登录 GitHub 账号，点击 "New app"
4. 选择 fork 的仓库，设置 Main file path: `app.py`
5. 在 "Advanced settings" 中添加环境变量：

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `API_BASE_URL` | API 服务地址 | `https://api.xiaomimimo.com/v1` |
| `API_KEY` | API 密钥 | `your_api_key_here` |
| `MODEL_NAME` | 模型名称 | `xiaomi-token-plan-cn/mimo-v2.5-pro` |

6. 点击 "Deploy"
